# ChronoVM Challenge

## Descripción
Un viejo programa llamado ChronoVM ha aparecido en un sistema abandonado. Parece un simple reloj digital, pero en realidad esconde un mecanismo de validación muy elaborado. Tu misión es descubrir cómo funciona y revelar el secreto que protege.

## Categoría
Reversing

## Dificultad
Insane

## Instrucciones
1. Ejecuta el binario `chronovm`
2. Resuelve los desafíos progresivos
3. Encuentra la clave de validación correcta
4. Obtén el flag final

## Notas
- El binario tiene protecciones anti-debugging activas
- Requiere conocimientos avanzados en reversing
- Tiempo estimado: 6-8 horas

¡Buena suerte!
