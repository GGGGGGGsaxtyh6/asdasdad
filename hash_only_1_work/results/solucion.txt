LECCIONES APRENDIDAS:
---------------------
1. Los binarios SUID deben usar rutas absolutas para ejecutar comandos
2. Nunca ejecutar comandos externos con system() usando rutas relativas
3. PATH hijacking es una técnica común para escalar privilegios
4. La variable PATH se puede manipular para ejecutar código arbitrario
5. Los binarios deben sanitizar el entorno antes de ejecutar comandos externos
6. La ubicación del binario no importa si la vulnerabilidad es la misma
